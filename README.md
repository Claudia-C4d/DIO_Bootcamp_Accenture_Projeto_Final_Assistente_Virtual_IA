DIO_Bootcamp_Accenture_Projeto_Final_Assistente_Virtual_IA
Projeto Final para o Bootcamp DIO -Accenture de uma assistente virtual com IA para finanças pessoais e investimentos.

- src - Contém o arquivo d eprogramação em Python Ai_Nila_app.py
- docs - Contém os 05 arquivos para documentação de cada etapa do projeto
- Pitch - Vídeo em MP4 com a apresentação do projeto em slides e com gravação de voz.
